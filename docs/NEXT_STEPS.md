# Fubako プロジェクト - 次のステップ

## 📊 現在の状況（Phase 1 完了）

### ✅ 完成している機能

#### 1. **Fubakoアプリ（Electron + Vue）**
- プロジェクト管理（開く、設定読み込み）
- コンテンツ管理（一覧、作成、編集、保存）
- 画像管理（アップロード、UUID リネーム、日付別フォルダ）
- プレビュー機能（Zola起動）
- 動的フォーム生成（10種類のフィールドタイプ）
- ターコイズ系フラットデザイン

#### 2. **サンプルサイト（Zola）**
- 企業サイト構成
- 3つのコンテンツタイプ（ニュース、導入事例、サービス）
- 充実したコンテンツ
  - ホームページ
  - 会社概要ページ
  - 3つのサービスページ（詳細）
  - 2つのニュース記事
  - 1つの導入事例
- プロフェッショナルなデザイン（シャープ、フラット）

#### 3. **ドキュメント**
- README.md
- ARCHITECTURE.md
- CONFIG.md

- PHASE1_COMPLETE.md

## 🎯 次のステップ（優先順位順）

### Phase 2A: 完成度向上（推奨：まずこれ）

#### 1. **画像の配置** 🎨
**優先度: 高**

現在、グレーのプレースホルダーになっている画像を配置します。

**必要な作業:**
```bash
# 画像を配置
sample-site/static/images/
├── hero-main.jpg
├── service-cloud.jpg
├── service-ai.jpg
├── service-security.jpg
├── news-cloudsync-pro.jpg
├── news-seminar.jpg
├── case-manufacturing.jpg
├── client-logo-global-manufacturing.jpg
├── logo.png
└── ogp-default.jpg
```

**所要時間:** 1-2時間（AI生成 + 配置）

#### 3. **バグ修正と改善** 🐛
**優先度: 高**

### Phase 2B: 機能拡張

#### 4. **画像リサイズ機能** 📐
**優先度: 中**

Sharp を使った画像の自動リサイズ機能を実装します。

**実装内容:**
- アップロード時に複数サイズを生成
- サムネイル、中サイズ、大サイズ
- WebP形式への変換（オプション）

**所要時間:** 4-6時間

---

#### 5. **メディアライブラリ** 🖼️
**優先度: 中**

アップロード済み画像を一覧表示・選択できる機能を追加します。

**実装内容:**
- 画像一覧表示
- 検索・フィルタリング
- 画像の選択・挿入
- 画像の削除

**所要時間:** 6-8時間

---

#### 6. **プレビューの改善** 👁️
**優先度: 中**

編集画面でのプレビュー機能を強化します。

**実装内容:**
- リアルタイムプレビュー（iframe埋め込み）
- プレビューの自動更新
- デスクトップ/モバイル表示切り替え

**所要時間:** 4-6時間

---

### Phase 2C: 高度な機能

#### 7. **Git連携** 🔄
**優先度: 低〜中**

バージョン管理とデプロイの自動化を実装します。

**実装内容:**
- Git初期化
- 自動コミット
- リモートリポジトリへのプッシュ
- GitHub Pages / Netlify / Vercel へのデプロイ

**所要時間:** 8-10時間

---

#### 8. **複数ユーザー対応** 👥
**優先度: 低**

複数人での編集を可能にします。

**実装内容:**
- ファイルロック機能
- 編集中の表示
- 競合の検出

**所要時間:** 10-12時間

---

#### 9. **プラグインシステム** 🔌
**優先度: 低**

カスタムフィールドタイプの追加を可能にします。

**実装内容:**
- プラグインAPI
- カスタムフィールドの登録
- プラグインの管理画面

**所要時間:** 12-15時間
