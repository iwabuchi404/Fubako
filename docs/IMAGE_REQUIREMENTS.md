# サンプルサイト用画像リスト

## デザイン方針
- **カラー**: ターコイズ系（#00BCD4, #26C6DA, #00ACC1）
- **スタイル**: スタイリッシュ、フラット、ミニマル
- **装飾**: シンプル、過度な装飾なし

## 必要な画像

### 1. ヒーロー画像（トップページ）
**ファイル名**: `hero-main.jpg`
**サイズ**: 1920x800px
**内容**: 
- モダンなオフィス空間、またはテクノロジー関連の抽象的なビジュアル
- ターコイズのアクセントカラー
- クリーンでプロフェッショナルな印象
- 人物は不要（または後ろ姿・シルエット程度）

### 2. サービスアイコン画像（3枚）
**ファイル名**: 
- `service-cloud.jpg` (クラウドソリューション)
- `service-ai.jpg` (AI・機械学習)
- `service-security.jpg` (セキュリティ)

**サイズ**: 800x600px
**内容**:
- フラットデザイン、アイコニック
- ターコイズをメインカラーに
- 各サービスを象徴するシンプルなビジュアル
- 背景は白またはライトグレー

### 3. ニュース記事サムネイル（2枚）
**ファイル名**:
- `news-cloudsync-pro.jpg` (新製品リリース)
- `news-seminar.jpg` (セミナー告知)

**サイズ**: 1200x630px
**内容**:
- **CloudSync Pro**: クラウドストレージ、データ同期のイメージ
- **セミナー**: プレゼンテーション、ビジネスミーティングのイメージ
- ターコイズのアクセント
- テキストスペースを考慮した構図

### 4. 導入事例 企業ロゴ（1枚）
**ファイル名**: `client-logo-global-manufacturing.jpg`
**サイズ**: 400x200px
**内容**:
- 架空の製造業企業のロゴ
- シンプルで信頼感のあるデザイン
- グレー系またはダークブルー系
- 「GLOBAL MANUFACTURING」の文字

### 5. 導入事例 メインビジュアル（1枚）
**ファイル名**: `case-manufacturing.jpg`
**サイズ**: 1200x800px
**内容**:
- 製造業の現場（工場、倉庫、在庫管理）
- タブレットやバーコードスキャナーを使用している様子
- クリーンで近代的な印象
- ターコイズのアクセント

### 6. 会社ロゴ（1枚）
**ファイル名**: `logo.png`
**サイズ**: 600x200px（透過PNG）
**内容**:
- 「株式会社テクノロジーイノベーション」のロゴ
- シンプルで読みやすい
- ターコイズをメインカラーに
- アイコン + テキストの組み合わせ
- 白背景でも使える

### 7. OGP画像（SNSシェア用）
**ファイル名**: `ogp-default.jpg`
**サイズ**: 1200x630px
**内容**:
- 会社名とタグライン「Innovation for Tomorrow」
- ターコイズのグラデーション背景（控えめ）
- シンプルで洗練されたデザイン
