# Zolaバイナリの配置について

このディレクトリには、Zolaの実行ファイルを配置してください。

## ダウンロード

https://www.getzola.org/documentation/getting-started/installation/

## 配置するファイル

- **Windows**: `zola.exe`
- **macOS**: `zola` (実行権限を付与: `chmod +x zola`)
- **Linux**: `zola` (実行権限を付与: `chmod +x zola`)

## バージョン

推奨バージョン: **0.18.0** 以降

## 確認方法

```bash
# Windows
.\bin\zola.exe --version

# macOS/Linux
./bin/zola --version
```

正常に動作すれば、バージョン情報が表示されます。
